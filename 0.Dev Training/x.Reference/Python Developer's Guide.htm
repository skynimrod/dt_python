Links for Developers

Python development takes place at SourceForge. You can find the following items there:

  • Python Project Page - CVS instructions, Bug Manager, Patch Manager, CVS browser, and more
  • Python Enhancement Proposals (PEPs) - Information on new language features, and some meta-information like release procedure and schedules
  • Developer FAQ - Tips on CVS, SSH and patch submission
  • Documentation development - Links and information related to the development of the documentation
  • Daily snapshots of the Python source tree
